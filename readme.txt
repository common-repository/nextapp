NextApp wordpress plugin provides an interface for mobile client to access the blog categories, articles and comments.

== Description ==

The Nextapp plugin allows you to make your blog go mobile. Once this plugin is installed all you have to do is create an app at [http://www.nextapp.cn](http://www.nextapp.cn) and publish it. Then sit back and reap the benefits of having your blog on the mobile scene!

Nextapp is an easy-to-use tool for anyone to create a mobile app for their website, with no programming knowledge necessary! Set up couldn't be more simple - if you can install a plugin to your WordPress website, you have all the skills you need. Then just choose your colors, upload your icons and header image, and you're done!


== Installation ==

After installation of NextApp WordPress plugin, activate it. Then visit [www.nextapp.cn](http://www.nextapp.cn) and start creating your mobile application for free.
